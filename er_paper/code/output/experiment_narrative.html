6. Takeaways

  1. The compilation pipeline works end-to-end: JSON → CPN → SMDP → solve. Fully general.
  2. Tabular RL has clear limits: works for simple fragments (Fig 2), fails for multi-stage processes (Fig 1) due to transient state spaces.
  3. Deep RL (PPO) finds policies no heuristic captures: 7.4% improvement by coordinating scheduling, routing, and assignment jointly.
  4. In healthcare ops, -7% cycle time is significant: patients spend substantially less time in the clinic — found automatically from the process model.
